# 적립 서비스 React Native 샘플

간단한 포인트 적립 앱을 위한 React Native(Expo) 구조 샘플입니다. 구글 로그인, 생체 인증 안내, 오퍼월 진행 현황, 하단 네비게이션 등을 포함한 기본 화면 구성을 제공합니다.

## 주요 기능
- 구글 로그인 트리거 및 로그인 후 생체 인증 등록 요청
- 적립 포인트, 배너, 오퍼월 플레이스홀더가 포함된 홈 화면
- 계정/포인트 확인 및 포인트 교환소 예약 안내
- 오퍼월 진행 현황 목록
- 설정(푸시, 광고 동의), 약관 진입, 로그아웃/회원탈퇴 액션

## 실행 방법
1. `npm install` 또는 `yarn` 으로 의존성을 설치합니다.
2. `npm start` 로 Expo 개발 서버를 실행합니다.
3. 필요 시 `npm run android`/`npm run ios`/`npm run web` 명령을 사용하세요.

> 실제 구글 인증 및 애드몹 오퍼월 연동은 SDK 추가 작업이 필요합니다.
